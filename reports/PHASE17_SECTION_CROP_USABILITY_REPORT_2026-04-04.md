# Phase 17 — 섹션 재정렬 + 슬롯 크롭 손맛 + 사용성/버그 보강

## 이번 패치의 목적
- 오른쪽 섹션 패널에서 카드 드래그로 순서를 바꿀 수 있게 한다.
- 슬롯 더블클릭 후 들어가는 크롭 모드를 키보드 중심에서 `직접 드래그 + 휠 줌` 중심으로 바꾼다.
- 업로드 자산 카드 드래그가 실제로 동작하도록 연결하고, 몇몇 잔버그를 정리한다.

## 실제 반영 내용
1. **섹션 카드 드래그 재정렬**
   - `renderSectionFilmstrip()` 카드에 `draggable=true` 추가
   - 섹션 카드 드래그 시작/오버/드롭/종료 이벤트 추가
   - 위/아래 버튼만 있던 재정렬을 `카드 직접 드래그` 방식으로 확장
   - 상/하 drop indicator 스타일 추가
   - 우측 섹션 패널 헤더 문구를 `드래그로 순서 변경` 기준으로 수정

2. **슬롯 더블클릭 크롭 모드 손맛 개선**
   - 크롭 모드 진입 시 기존 transform을 파싱해서 이어서 편집 가능
   - 크롭 모드 중 슬롯 내부 pointer drag로 이미지 팬(pan) 가능
   - 크롭 모드 중 wheel 줌 가능
   - 크롭 중에는 일반 선택/이동보다 크롭 팬이 우선되도록 이벤트 우선순위 수정
   - 퀵바에 `적용 / 초기화 / 취소` 버튼 추가
   - 크롭 모드에서는 공통 액션 대신 크롭 전용 액션만 보이도록 퀵바 범위 조정
   - mini HUD가 크롭 배율을 보여주도록 보강

3. **업로드 이미지 카드 드래그 실동작 보강**
   - 누락되어 있던 `renderUploadAssetLibrary()`를 소스에 정식 추가
   - 자산 카드에 `draggable=true`, ref/label data 속성, 썸네일/메타 출력 추가
   - 클릭으로 현재 선택 슬롯에 적용, 드래그로 슬롯 드롭 적용 흐름 유지

4. **잔버그 수정**
   - 섹션 카드 렌더러의 중복 `<strong>` 제거
   - source/bundle 재동기화 후 재빌드
   - phase8 validate 재실행

## 검증
- `python3 scripts/build_local_bundle.py`
- `node --check app.bundle.js`
- `python3 scripts/validate_phase8.py`
- 결과: 167 / 167 통과

## 미리캔버스급 사용성(디테일) 패치 우선순위

### A. 다음 스프린트에서 바로 체감 나는 것
1. **섹션 필름스트립 다중선택 + 다중 재정렬**
   - 미리캔버스는 페이지 리스트에서 다중 선택 후 순서를 바꿀 수 있다.
   - 우리 앱도 Shift/Ctrl 기준 섹션 다중선택과 묶음 이동이 필요하다.

2. **섹션 카드 `더보기(···)` 메뉴**
   - 이름 바꾸기 / 복제 / 위로 / 아래로 / 삭제 / 메모
   - 카드 hover 시만 노출

3. **슬롯 크롭 오버레이 개선**
   - 현재는 퀵바 + 키보드/휠 중심이다.
   - 다음 단계는 슬롯 내부에 crop grid, 배율 HUD, 적용/취소 floating chip 추가가 필요하다.

4. **슬롯 이미지 잠금**
   - 프레임 안 사진 잠금 개념처럼, 슬롯 안 이미지를 고정해서 드래그 교체를 막는 옵션이 필요하다.

5. **좌측 속성 패널 문맥형 축소**
   - 현재 속성 패널은 여전히 많은 제어가 섞여 있다.
   - 선택 타입별로 핵심 카드만 보이고 나머지는 `고급` 접이식으로 더 줄여야 한다.

### B. 에디터 디테일을 끌어올리는 것
6. **선택 퀵바의 버튼 수를 더 줄이고 `…` 메뉴 도입**
7. **오브젝트 우클릭 컨텍스트 메뉴**
   - 복제 / 삭제 / 잠금 / 앞으로 / 뒤로 / 그룹 / 크롭
8. **레이어 패널 드래그 z-order**
9. **스냅 가이드에 거리 라벨 표시**
10. **정렬/간격 액션을 퀵바에서 아이콘형으로 단순화**
11. **텍스트 편집 시 상단 작은 inline toolbar**
12. **이미지/텍스트 더블클릭 진입 애니메이션 및 상태 강조**
13. **패널 상태/폭/마지막 탭 복원 안정화**
14. **선택 영역 PNG의 bbox 미리보기**
15. **내보내기 전 체크를 toast + 상세 펼침 구조로 재설계**

### C. 미리캔버스식 고급 편집감으로 가는 것
16. **요소별 mini transformer 개선**
   - 회전, 비율 유지 resize, Option/Alt 기준 center resize
17. **키보드 단축키 체계 재설계**
   - 복제, 잠금, 앞으로/뒤로, 그룹, 크롭 진입/적용, 섹션 이동
18. **실제 썸네일 캐싱**
   - 현재는 DOM clone 축소 렌더링이다.
   - 장기적으로는 저해상도 이미지 캐시를 만들면 스크롤이 더 부드러워진다.
19. **도구/속성 패널 검색**
   - 명령 팔레트와 별개로 속성 검색 지원
20. **작업중 피드백 체계**
   - 저장 완료, 적용 완료, 오류, 자동 복구, 내보내기 진행 상태를 통일된 toast 시스템으로 재설계

## 결론
이번 패치는 `섹션은 카드로 직접 옮기고`, `크롭은 드래그/휠로 만진다`는 최소한의 손맛을 만든 단계다.
다음 점프는 **섹션 카드 더보기 메뉴 + 크롭 오버레이 시각화 + 이미지 잠금**이 가장 효율적이다.
