# SPEC

## 제품 정체
이 프로젝트는 서버 SaaS가 아니라, 로컬에서 `index.html`을 `file://`로 직접 열어 쓰는 **HTML 기반 상세페이지 편집기**다.

## 핵심 목표
- 기존 상세페이지 HTML을 연다.
- 이미지 슬롯/텍스트/섹션을 감지하고 편집한다.
- 코드 편집과 캔버스 편집을 왕복한다.
- 기본 저장은 `linked` 패키지로 한다.
- 필요하면 `embedded` HTML도 내보낸다.
- 긴 상세페이지를 빠르게 탐색/편집/내보낸다.

## 비목표
- 서버 배포형 협업 SaaS
- 일반 문서 편집기
- 순수 디자인 툴 대체

## 완료의 정의
- fixture 회귀가 없다.
- 대형 이미지/문서에서 편집이 버틸 수준으로 유지된다.
- linked 저장 후 재오픈과 embedded 저장이 모두 동작한다.
- HTML↔캔버스 왕복이 명확하게 추적된다.
