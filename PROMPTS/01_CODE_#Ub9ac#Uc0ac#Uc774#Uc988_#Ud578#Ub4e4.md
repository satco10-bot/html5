선택된 요소에 대한 리사이즈 핸들을 구현해줘.

Goal:
- 슬롯, 박스, 텍스트 요소를 마우스로 크기 조절할 수 있게 만든다.

Context:
- 현재 선택/레이어/저장/export 흐름을 먼저 읽고, 기존 구조를 최대한 유지한다.
- 저장 후 다시 열어도 크기 변경이 유지되어야 한다.

Constraints:
- 로컬 전용 정적 웹앱이어야 한다.
- index.html을 file://로 직접 열어도 동작해야 한다.
- 서버/도메인 의존 기능 추가 금지
- uploaded: 및 상대경로 이미지 처리 보존
- fixture 회귀 금지

Done when:
- 드래그로 크기 조절 가능
- 저장 후 다시 열어도 유지
- 관련 검증 또는 체크 추가
- 변경 파일 요약과 남은 위험 보고
