직접 편집 핵심 기능을 추가해줘.

Goal:
- 선택 요소 복제
- 선택 요소 삭제
- 새 텍스트 추가
- 새 박스 추가
- 새 이미지 슬롯 추가

Context:
- 미리캔버스/Figma처럼 캔버스 위에서 빠르게 구조를 편집하는 흐름이 목표다.
- 새로 추가한 요소도 레이어 패널, 저장, 재오픈, export 흐름에 자연스럽게 포함되어야 한다.

Constraints:
- 로컬 전용 정적 웹앱 유지
- file:// 직접 실행 유지
- uploaded: 및 fixture 회귀 금지
- 런타임 오버레이가 저장 결과에 섞이면 안 됨

Done when:
- 버튼 또는 단축키로 복제/삭제 가능
- 새 텍스트/박스/슬롯을 생성 가능
- 새 요소가 선택, 레이어, 저장 흐름에 포함됨
- 변경 파일과 테스트 결과 요약
