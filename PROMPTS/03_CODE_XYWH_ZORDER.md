정밀 편집 기능을 추가해줘.

Goal:
- 선택 요소의 X / Y / W / H 숫자 편집 패널
- 앞으로 가져오기 / 뒤로 보내기
- 이미지 nudge 또는 위치 미세조정

Context:
- 현재 앱은 대략적인 이동은 가능하지만, 실전 상세페이지 편집용 정밀 편집이 부족하다.
- 저장 후 다시 열어도 동일한 상태가 유지되어야 한다.

Constraints:
- 로컬 전용 정적 웹앱 유지
- file:// 직접 실행 유지
- 저장/재오픈/export 경로를 함께 고려할 것
- fixture_05 회귀 금지

Done when:
- 숫자 입력으로 위치/크기 조절 가능
- z-order 변경 가능
- 저장 후 상태 유지
- 변경 요약과 남은 이슈 보고
