스타일 컬러 스튜디오 · HTML 색상 빠른 교체

    

    style 블록과 inline style에 직접 적힌 색상을 모아서 한 번에 바꿉니다. 문서 전체 또는 현재 선택 섹션 범위로 좁힐 수 있습니다.
    

    


    



    

    

    style 블록과 inline style에서 색상을 읽어오면 여기 표시됩니다.
    

    

    

    

    
CSS 변수 / 컬러 토큰 · HTML 특화

    

    :root와 style 블록에 있는 컬러 토큰을 한 번에 수정합니다. 토큰은 전체 테마를 바꾸고, 위 컬러 스튜디오는 개별 스타일 색상까지 다룹니다.
    

    



    

    

    

    

    
typography / spacing / radius / shadow 토큰 · 확장 편집기

    

    컬러 외에 글꼴/간격/라운드/그림자 토큰도 한 번에 수정합니다. HTML/CSS에 직접 박힌 디자인 토큰을 빠르게 조정할 때 유용합니다.
    

    



    

    디자인 토큰을 읽으면 여기 요약이 표시됩니다.
    

    

    

    

    
반복 inline 색상 → CSS 변수 추출 · 자동 정리

    

    inline style에 반복된 색상을 찾아 CSS 변수로 승격합니다. 동일 색이 여러 슬롯/박스에 박혀 있을 때 유지보수가 쉬워집니다.
    

    



    

    반복 inline 색상이 있으면 여기서 CSS 변수로 정리할 수 있습니다.
    

    

    

    

    
섹션별 테마 팔레트 · 선택 섹션 우선

    

    선택한 섹션에서 많이 쓰는 색을 팔레트로 묶어 보여줍니다. 상세페이지 섹션별 톤 조정이나 CTA/배지 색 수정에 유용합니다.
    

    



    

    섹션을 선택하면 테마 팔레트를 읽어옵니다.
    

    

    

    

    
슬롯 schema / 라벨 · 자동 생성

    

    현재 감지된 슬롯에 섹션 기준 라벨과 schema를 자동 부여합니다. 저장 HTML에는 data-slot-label, data-slot-schema, data-image-slot이 같이 남습니다.
    

    


    

    슬롯을 감지하면 여기서 자동 schema를 만들 수 있습니다.

깨진 자산 경로 일괄 relink

    

    미해결 자산을 파일명 기준으로 한 번에 다시 연결합니다. 매칭된 자산은 data URL로 치환해 저장 HTML에도 남습니다.
    

    



    

    

    미해결 자산이 있으면 여기서 다시 연결할 수 있습니다.
    

    

    

    

    
마켓 업로드용 HTML lint

    

    스크립트/iframe, 고정 포지션, 원격 리소스, placeholder 등 마켓 업로드에서 문제가 되기 쉬운 항목을 별도로 검사합니다.
    

    


    

    

    

    

    

    
색상 대비 / 가독성 lint

    

    현재 미리보기 기준으로 텍스트/배경 대비가 낮은 요소를 찾아줍니다. 선택 섹션 점검 전에 한 번 돌려보면 눈에 띄지 않는 가독성 문제를 빨리 잡을 수 있습니다.
    

    


    

    가독성 문제가 있으면 여기서 바로 확인할 수 있습니다.
